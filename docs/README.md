# Documentación

Este directorio agrupa toda la información vigente del proyecto. Cada archivo reemplaza a las guías antiguas que estaban sueltas en la raíz.

| Archivo | Contenido |
| --- | --- |
| `FRONTEND.md` | Arquitectura del sitio, estructura de carpetas y módulos disponibles. |
| `COLOR-PALETTES.md` | Sistema de paletas, selector y convenciones de diseño. |
| `MI-MESA.md` | Flujo completo de la vista *Mi Mesa*, imágenes locales y subida de fotos. |
| `GOOGLE-DRIVE.md` | Resumen de la integración serverless (Netlify Function + Drive). |
| `DEPLOYMENT.md` | Guía de desarrollo local y despliegue en Netlify (sitio + Functions). |
