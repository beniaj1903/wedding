═══════════════════════════════════════════════════════════════
  ⚡ IMPORTANTE: SERVIDOR LOCAL REQUERIDO
═══════════════════════════════════════════════════════════════

Los componentes HTML se cargan dinámicamente, necesitas servidor:

Opción 1 - Python:
    python3 -m http.server 8000
    Abre: http://localhost:8000

Opción 2 - Node.js:
    npx http-server -p 8000
    Abre: http://localhost:8000

Opción 3 - VS Code:
    Instala extensión "Live Server"
    Click derecho → "Open with Live Server"

❌ NO funcionará con file:///

═══════════════════════════════════════════════════════════════
  🆘 SOLUCIÓN DE PROBLEMAS
═══════════════════════════════════════════════════════════════

Problema: "Página en blanco"
→ Verifica que uses servidor local (no file://)
→ Abre consola (F12) y busca errores

Problema: "Componentes no cargan"
→ Verifica rutas en component-loader.js
→ Asegúrate que archivos existan en components/

Problema: "Estilos no se aplican"
→ Verifica links CSS en index.html
→ Limpia caché del navegador (Cmd+Shift+R)

Problema: "JavaScript no funciona"
→ Espera a que componentes carguen
→ Revisa consola para errores

Mensaje correcto en consola:
✅ Componentes HTML cargados
✅ Aplicación inicializada correctamente
